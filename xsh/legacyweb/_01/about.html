About the website:

Website made by NDA


This website is made to be a personal blog site where I post updates and news about my projects and share my new side projects.
The website contains different side projects I do including games, 3D models and different coding side projects.
The website design is inspired by the early WEB 2.0 websites with experimental UX and UI.

The website is coded using only HTML and CSS as it's building blocks.



All external links leading to different websites are not affiliated NDA, NDA Soft or different previous aliases mentioned throughout the website.
Certain images and video files are hosted on Discord and as a result can impact the loading performance of the website for slower connections.
