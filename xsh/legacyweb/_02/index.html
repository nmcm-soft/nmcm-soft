Website Update

New website facelift!

I've made a new sleek design keeping in style with the early WEB 2.0 aesthetic. This design is a huge improvement over the first one. The curvaceous svgs that don't scale properly are now gone in favour of a more simple and more aesthetically pleasing layout and visuals. The website content is now in a wrapper, making it easily scalable for different devices and screen resolutions without destroying the layout.

The Liminal Scope

Updated to v0.4!

The Liminal Scope has undergone a lot of changes during it's 6 months of development.
This update brings QOL changes and a range of improvements from previous versions;

  • Added in-game settings tab
  • Added graphics settings
  • Added resolution settings
  • Made the game more accessible to lower end machines
  • Added FOV sliders and constant mouse sensitivity
  • Optimized "The Complex" map
  • Optimized "Parking Garage" map
  • Fixed light glitches for "The Complex" & "Parking Garage"
  • Added minor aesthethical improvements to "The Complex"
  • Added an otherwordly entity to "The Complex"

This update also adds a new map - "Liminal Subway".
Explore an abandoned subway that branches into side corridors. Stumble through once bustling abandoned corridors with nothing more than distant humming as company. There is a faint sound of metal banging... is anyone there?
